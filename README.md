# Pathing Category Explorer Plugin - Blish HUD Module
Extend the context menu in the Pathing Category Explorer. 

Must be enabled after the Pathing module is enabled.
## Features
* Select node recursively.
* Deselect node recursively.
* Select the node and deselect all others.
* Fix node expansion bug.

## Usage
1. Open the Pathing Category Explorer. 
2. Right-click a node.

## Download
From [Releases](https://github.com/cy-sp-howard/PathingCategoryExplorerPlugin/releases) page.
## Install Guide
1. Right-click Blish-HUD icon from system tray where bottom right.
2. Left-click "Open settings folder".
3. Paste [PathingCategoryExplorerPlugin.bhm](https://github.com/cy-sp-howard/PathingCategoryExplorerPlugin/releases) file into "modules" folder.
4. Restart Blish-HUD.
5. Click the Blish-HUD icon in the game then Left-click the `PathingCategoryExplorerPlugin` tab under `Manage Modules` section.
6. Click the `Enable` button for enable `PathingCategoryExplorerPlugin` module.

More detail from Blish-HUD Offical [Tutorial](https://blishhud.com/docs/user/guides/installing-modules/#manually-installing-modules).
